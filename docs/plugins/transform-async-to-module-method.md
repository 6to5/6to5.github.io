## Installation

```sh
$ npm install babel-plugin-transform-async-to-module-method
```

## Usage

Add the following line to your `.babelrc` file:

```json
{
  "plugins": [
    ["transform-async-to-module-method", {
      "module": "bluebird",
      "method": "coroutine"
    }]
  ]
}
```
