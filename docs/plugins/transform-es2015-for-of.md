This plugin transforms ES2015 for of loops to ES5.

## Installation

```sh
$ npm install babel-plugin-transform-es2015-for-of
```

## Usage

Add the following line to your `.babelrc` file:

```json
{
  "plugins": ["transform-es2015-for-of"]
}
```
