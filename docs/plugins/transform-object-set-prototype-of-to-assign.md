## Installation

```sh
$ npm install babel-plugin-transform-object-set-prototype-of-to-assign
```

## Usage

Add the following line to your `.babelrc` file:

```json
{
  "plugins": ["transform-object-set-prototype-of-to-assign"]
}
```
